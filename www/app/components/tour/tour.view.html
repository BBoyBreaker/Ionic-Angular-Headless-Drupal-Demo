NG Drupal7 Services

Native app usecases with angular, ionic and cordova

Detecting first visit

First visti flag is stored on device and skips tour on second fisit automatically

Registration and redirects

Registered flag is stored on device and redirects if the user is already registered

Easy to implement Authentication

Authentication is done over a Authentication Service for simplifying the workflow

Client and server validation for forms

Form are setup with with ngMessages directive and handles server side error messages

Reauthentication on app start

On app start the authentication state gets refreshed and become either authed or not

Content feed

Nodes are listed over a view that implements refresh and infinite scroll functionality.

CRUD operations

Nodes can be created edited and deleted

Camera usage

Take images over the devices camera (or cordova mocks plugin on chrome) and upload attache them to a node.

Restrictions by role

CRUD operations are restricted over the user roles

Route access

Handle route access over access levels and roles

Easy to use directives

For now a validation helper and a ui-interactor directive are provided
